# **App Name**: FlutterFolio Pro

## Core Features:

- Profile Showcase: Showcase experience, skills, and a professional summary as a Flutter developer with 5+ years experience.
- Project Portfolio: Display featured projects with descriptions, images, and links to GitHub repositories or live demos.
- AI-Powered Resume Enhancement: Use an AI tool to analyze resume text and provide suggestions for improvement, such as optimizing keywords or rephrasing descriptions for maximum impact.
- Skills Visualization: Present skills using charts or graphs to visually represent proficiency levels.
- Contact Form: Allow potential clients or employers to send messages directly through the portfolio.
- Admin Panel: A password-protected admin panel for managing the content of the portfolio, including projects, skills, and personal information.
- Blog Section: Ability to create blog posts to show subject matter expertise with markdown editor and preview, demonstrating expertise and thought leadership in Flutter development.

## Style Guidelines:

- Primary color: Deep purple (#6750A4) to evoke sophistication and innovation in the design.
- Background color: Light purple (#F2F0F9), subtly desaturated for a clean and professional look.
- Accent color: Soft lavender (#D0BCFF) for highlighting key elements and interactive components.
- Body text and headlines: 'Inter', a sans-serif font known for its modern and neutral appearance suitable for both headings and body text.
- Use a consistent set of line icons to represent skills, technologies, and social media links.
- Implement a clean, responsive layout with clear sections for easy navigation and readability on all devices.
- Incorporate subtle animations and transitions to enhance the user experience, such as fade-in effects or hover animations on project cards.
